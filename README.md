# Vending machine


## Getting Started

### api
```bash
cd api && npm install
```

Please add .env file

```
PORT=****
ACCESS_TOKEN_SECRET=**********
```

Run the development server:

```bash
npm run serve
```

#### Endpoints
##### GET

/product

/product/:id

/user

/user/login

#### POST

/user create user

/product create/delete product

/deposit 

/user/reset

/buy

/user/login

### web
```bash
cd web && npm install
```

Run the development server:

```bash
npm run dev
```
