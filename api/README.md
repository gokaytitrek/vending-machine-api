# Vending machine api


## Getting Started

First, install the dependencies:
```bash
npm install
```

Please add .env file

```
PORT=****
ACCESS_TOKEN_SECRET=**********
```

Run the development server:

```bash
npm run serve
```

### Endpoints
#### GET

/product

/product/:id

/user

/user/login

#### POST

/user create user

/product create/delete product

/deposit 

/user/reset

/buy

/user/login
